int  color

Memory registration.

Registers the specified memory region to global memory and returns an address translation key for it. The color that will be used for GMA with the address is also included in the key.

Parameters
addrLogical address of the top of the memory region to be registered.
sizeSize of the region to be registered.
colorColor number that will be used for GMA with the global memory.
Return values
ACP_ATKEY_NULLFail.
otherwiseAddress translation key.
